# DS image viewer

DS Capture와 연동되거나 독립적으로 동작하는 고성능 이미지 뷰어 프로젝트입니다.

## 🚀 프로젝트 개요
- **목적**: 빠르고 가볍지만 강력한 편집 기능을 갖춘 이미지 뷰어 개발
- **주요 기술**: Python, Tkinter (또는 고급 UI 라이브러리), Pillow (이미지 처리)
- **핵심 기능**:
    - 대용량 이미지의 빠른 로딩 및 렌더링
    - 마우스 휠을 이용한 부드러운 확대/축소 및 이동
    - 기본적인 이미지 보정 및 필터 기능
    - DS Capture와의 연동 (최근 캡처 이미지 즉시 보기 등)

## 📅 개발 로드맵
- [ ] 1단계: 기본 UI 프레임워크 및 이미지 로딩 엔진 구축
- [ ] 2단계: 이미지 조작(확대/축소, 회전, 이동) 로직 구현
- [ ] 3단계: 고급 편집 및 필터 기능 추가
- [ ] 4단계: 라이센스 인증 시스템 및 배포 최적화

## 🛠️ 개발 환경
- **Language**: Python 3.14+
- **Build Tool**: Nuitka (보안 빌드)
- **License System**: DS 계열 공통 라이센스 규격 준수
