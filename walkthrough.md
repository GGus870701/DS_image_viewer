### [v2.00.01] - UI/UX 고도화 및 GPS 연동 강화
- **GPS 기능 최적화**: 
    - 하단 GPS 전용 섹션을 제거하고 메인 EXIF 테이블 하단에 위도/경도 통합.
    - 네이버/카카오/구글 지도 버튼을 메인 툴바로 이동 (GPS 데이터 존재 시에만 동적 노출).
- **분할 뷰(Split View) 사용성 혁신**:
    - **1:1 리셋**: 중앙 구분 바(Handle) 더블 클릭 시 즉시 1:1 비율 복구 기능 구현.
    - **비교 UX**: 분할 모드 전환 시 오른쪽 화면에 자동으로 다음 이미지를 로드하여 비교 편의성 증대.
    - **정밀 제어**: '화면 맞춤', '회전', '반전' 등 툴바 기능이 현재 포커스 된 창(Active Viewport)에만 적용되도록 연동.
- **UI/UX 디테일링**:
    - **상태바(Status Bar)**: 파일 경로, 해상도 항목에 직관적인 아이콘 추가 및 레이아웃 정렬 최적화.
    - **작업 표시줄**: `AppUserModelID` 설정을 통해 독립된 아이콘으로 표시되도록 Windows 최적화.
    - **디자인 일관성**: 툴바 내 지도 버튼들의 높이와 너비(50px)를 균일화하여 시각적 완성도 제어.
- **시스템 안정성**:
    - 분할 뷰 전환 시 네비게이터(미니맵)가 활성 창의 썸네일과 위치를 정확히 추적하도록 동기화 로직 보정.
    - 정보 패널 내 테이블 열 너비를 고정하여 데이터 길이에 따른 UI 흔들림(Jitter) 방지.
